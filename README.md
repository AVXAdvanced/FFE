# FFE Development Beta Branch

The "dev_beta" branch of FFE houses the newest working Internal Beta. 
These builds come uncompiled, and may not be stable and/or fully functional.

If you are just willing to encrypt some files, and aren't a developer, you shouldn't use these builds.

## Compiling

To compile these builds of FFE (and turn them into a ".exe" file, use this command (on windows):

"pyinstaller --onefile {"File Path, with paranthesis. You may also drag the file here from explorer"}"
